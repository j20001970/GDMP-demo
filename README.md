# GDMP Demo
This is the demo project for [GDMP](https://github.com/j20001970/GDMP) plugin.

**Warning**: This repository is supposed to be rebased and squashed constantly.

Current Godot version: **3.6.stable**

Prebuilt libraries are no longer provided in `3.x` branch. We encourage users migrating to Godot 4.x if possible.
This branch now serves as a reference setup of Godot 3.x project.
